SimplePrivacy is a simple java software provides encryption/decryption and message digest functionalities with common algorithms.

Main features:
A swing GUI to facilitate operations;
AES128/CBC and AES/ECB cipher algorithms;
MD5, SHA1 and SHA256 message digest algorithms.

It generates and verifies message digests in format compatible to gnu tools md5sum, sha1sum and sha256sum.

Usage:
Just type "java -jar simpleprivacy-0.5.0.jar" to run this application.

Compilation:
Make sure you have ant installed;
Enter the source directory which contains build.xml and type "ant";
That's all. The generated jarball is in the "dist" directory.

Java platform requirement: Java 8 (jdk/jre 1.8) or above
